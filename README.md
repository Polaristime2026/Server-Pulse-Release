# ServerPulse Release

Public release assets for token-free ServerPulse test installs.

## Install

```bash
curl -fsSL https://github.com/Polaristime2026/Server-Pulse-Release/releases/download/v0.1.3/install-panel.sh \
  | sudo env SERVERPULSE_SOURCE_URL=https://github.com/Polaristime2026/Server-Pulse-Release/releases/download/v0.1.3/serverpulse.tar.gz bash
```

This release includes low-memory install protection for 1C1G test servers.
